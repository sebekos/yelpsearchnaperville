Requries Node.js

1. Clone repo "git clone git@github.com:sebekos/yelpsearchnaperville.git"
2. Change directory "cd yelpsearchnaperville"
3. Install dependencies "npm install react react-dom react-router-dom redux react-scripts react-redux redux-thunk"
4. Add .env file "touch .env"
5. Inside the .env file add the following line:
	YELP_API_KEY=(Your yelp API Key)
6. Start "npm run dev"
